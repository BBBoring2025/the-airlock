# SHA256SUMS (Release Asset)

Bu dosya repo içinde *bilgilendirme amaçlıdır*.

✅ Doğru doğrulama yöntemi:
- Release paketinin (the-airlock-vX.Y.Z-final.zip) SHA-256 değeri,
  GitHub Release'de ayrı bir asset olarak yayınlanan `SHA256SUMS.txt` dosyasıyla
  doğrulanmalıdır.

⚠️ Not:
- Aynı zip'in içinde "kendi checksum'u" tutulursa self-referential olur ve
  pratikte güvenilir doğrulama sağlamaz.
- Bu yüzden checksum dosyası zip'in *dışında* yayınlanır.

Örnek komut:
  shasum -a 256 the-airlock-vX.Y.Z-final.zip > SHA256SUMS.txt
